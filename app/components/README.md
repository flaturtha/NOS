# app/components

This directory contains all React components used throughout the NOS landing page system. 

- Each component should be mobile-first, accessible, and senior-friendly.
- Shared UI elements (buttons, dividers, etc.) can be placed in the `ui/` subdirectory.
- Page section components (e.g., OfferSection, BookGrid, TestimonialsSection) should be kept modular for easy reuse across landing page variants.

Please keep this README updated with any conventions or architectural notes relevant to this folder.
